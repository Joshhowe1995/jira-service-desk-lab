# Ticket Handling – Jira Service Desk Lab

This section demonstrates hands-on handling of common IT service desk requests using Jira Service Management.  
Each ticket follows a realistic lifecycle from user submission through investigation to resolution.

---

## 01–02: Forgotten Password Ticket
01 – User submits a forgotten password request via the service portal  
02 – Ticket is processed, user identity verified, and password reset completed

Demonstrates:
- User request intake
- Identity verification
- Secure resolution of access issues
- Ticket status progression

---

## 03–04: Slow Computer Issue
03 – User reports performance issues with their workstation  
04 – Issue investigated, resolved, and ticket closed

Demonstrates:
- Incident triage
- Basic troubleshooting workflow
- Resolution documentation
- Clear closure of incidents

---

## 05–06: New Employee Onboarding
05 – New employee onboarding request submitted  
06 – Onboarding request completed and marked as resolved

Demonstrates:
- Service request handling
- Structured onboarding workflow
- End-to-end request completion
